# etherStarter

### Install node packages and run the server:

```
npm install
node server.js
```

### Build the project:

```
./build
```

### Run eth client:

```
./start_eth
```

### Visit `http://localhost:3000`

Remember to recompile `etherstart_create.html_template` by `tools/mkcreate` or `./build` after changes to that file.
